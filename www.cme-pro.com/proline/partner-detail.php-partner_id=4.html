Kissbox
 

 

In cooperation with CME, one of the worlds leading MIDI-keyboard manufacturers Kiss-Box will introduce its extension board for the UF range of Midi keyboards at the Frankfurt Prolight and Sound exhibition. With the extension board the CME keyboards will have the capability to transmit Midi data directly via a connected Ethernet network using Kiss-Box RTP-MIDI technology.

Specifications:
- compatible with the complete UF full range MIDI keyboard (including UF 1st Generation and 2nd Generation)
- compatibility with the new VX range currently in study
- installed in the expansion slot of the keyboard
- no external power required (powered by the keyboard)
- no functionnality lost in the keyboard
- configuration using local DIP-Switches or via Kiss-Box Windows editor
- DIP switches allows to declare up to four CME keyboards on the same network
- the Kiss-Box Editor allows to declare any number of CME keyboards on the network
- fully RTP-MIDI compatible (thus compatible with Kiss-Box range of product, Win XP driver and Mac OSX)
- the CME-NET boards is seen by computer as two MIDI IN and two MIDI OUT ports (one the first MIDI IN is the keyboard itself, the other ports are standard DIN connectors)
- no boot up time (keyboard immediately available when powered on)
- extremely low latency (less than 900 microseconds)
- CME-NET does not need a computer on the network to operate (peer to peer operation with a RTP MIDI Kiss-Box is possible)
