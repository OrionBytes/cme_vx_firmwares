U2MIDI USB Cable-Like MIDI Interface

U2MIDI, the world’s first USB integral cable-like MIDI interface, is a MIDI-to-USB device, providing 1 MIDI-IN, 1 MIDI-OUT. It supports plug-and-play and is compatible with both Macintosh and Windows systems.

U2MIDI also includes a function which allows multiple units to be used simultaneously. Thanks to its special processing technology, U2MIDI provides faster response and easiest way to connect any MIDI device to the computer.

Features:

- Integral cable-like USB MIDI interface providing 1 MIDI-IN, 1 MIDI-OUT, USB 2.0 full speed transmission
- Faster response than normal USB MIDI interface
- MIDI-to-USB plug-and-play, USB class-compliant
- USB bus powered, with LED indicator

System Requirements:

- PC with Windows XP/VISTA/Win 7 32 bit 
- Mac OS 9.0.4 or higher
- Computer with USB port
